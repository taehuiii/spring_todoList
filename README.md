# 🗒️ Todo List 🗒️ 

***

#### [ 기능 소개 ]
- 할일 카드를 `작성` / `조회` / `수정` / `삭제` 할 수 있습니다.
- 할일 카드를 완료처리할 수 있습니다.
- 댓글을 `작성` / `수정` / `삭제` 할 수 있습니다.
- 할일 목록을 오름차순/내림차순으로 정렬할 수 있습니다.
- 할일 목록을 작성자 기준으로 필터링할 수 있습니다.
- 할일 작성 시 글자수를 검사할 수 있습니다.
